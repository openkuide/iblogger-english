# Context Clues, Prefixes, & Suffixes (ការយល់ពាក្យតាមបរិបទ និងបុព្វបទ-បច្ច័យបទ)

ការទន្ទេញពាក្យម្តងមួយៗគឺមិនគ្រប់គ្រាន់ទេសម្រាប់ការអានអត្ថបទវែងៗ។ ផ្ទុយទៅវិញ ការរៀនស្វែងយល់ពីតម្រុយក្នុងបរិបទ (Context Clues) និងផ្នែកផ្សំរបស់ពាក្យ (Prefixes/Suffixes) ជួយឱ្យយើងអាចទាយអត្ថន័យនៃពាក្យថ្មីៗបានយ៉ាងងាយស្រួល។

Memorizing word lists is inefficient for long-term reading. Instead, using context clues and understanding word parts (prefixes/suffixes) allows you to deduce the meaning of unfamiliar words dynamically.

---

## 🔍 ១. ការទាយន័យពាក្យតាមបរិបទ (Context Clues)

នៅពេលជួបពាក្យដែលមិនស្គាល់ កុំទាន់អាលបើកវចនានុក្រម។ ព្យាយាមរកមើល៖
* **Synonyms (ពាក្យន័យដូច):** ជួនកាលអ្នកសរសេរពន្យល់ពាក្យពិបាកដោយប្រើពាក្យងាយស្រួលនៅជិតនោះ។
  - *ឧទាហរណ៍៖* The application was **sluggish**; indeed, it was very **slow**. (ពាក្យ *sluggish* មានន័យថា *slow*)
* **Antonyms (ពាក្យន័យផ្ទុយ):** ស្វែងរកពាក្យផ្ទុយដើម្បីទាយន័យពាក្យនោះ។
  - *ឧទាហរណ៍៖* Unlike the **unstable** legacy database, this new cloud service is completely **robust**. (ពាក្យ *robust* មានន័យផ្ទុយនឹង *unstable* ពោលគឺរឹងមាំ ឬមានលំនឹង)
* **Examples (ឧទាហរណ៍):**
  - *ឧទាហរណ៍៖* High latency caused several **anomalies**, such as connection dropouts, data corruption, and slow response times. (ពាក្យ *anomalies* មានន័យថា ភាពមិនប្រក្រតី ឬបញ្ហាខុសធម្មតា)

## 🧱 ២. បុព្វបទ និងបច្ច័យបទ (Prefixes & Suffixes)

ពាក្យអង់គ្លេសជាច្រើនត្រូវបានបង្កើតឡើងដោយការបន្ថែមបុព្វបទ (Prefix - នៅខាងមុខ) ឬបច្ច័យបទ (Suffix - នៅខាងក្រោយ) ទៅលើពាក្យឫស (Root Word)។

### ក. Prefixes (បុព្វបទ - ប្តូរអត្ថន័យពាក្យ)

| Prefix | អត្ថន័យ (Meaning) | ពាក្យឫស (Root) | ពាក្យថ្មី (New Word) & អត្ថន័យ |
| :--- | :--- | :--- | :--- |
| **un- / dis-** | មិន / ផ្ទុយ (not/opposite) | clean, connect | **un**clean, **dis**connect (មិនស្អាត, ផ្តាច់ការតភ្ជាប់) |
| **re-** | ម្តងទៀត (again) | build, write | **re**build, **re**write (សាងសង់ឡើងវិញ, សរសេរឡើងវិញ) |
| **pre-** | មុន (before) | view, configure | **pre**view, **pre**configure (មើលជាមុន, កំណត់រចនាសម្ព័ន្ធជាមុន) |
| **post-** | ក្រោយ (after) | deploy, script | **post**-deploy, **post**script (ក្រោយការដាក់ឱ្យដំណើរការ, សេចក្តីបន្ថែម) |
| **sub-** | ក្រោម / រង (under/lower) | net, folder | **sub**net, **sub**folder (បណ្តាញរង, ថតរង) |

### ខ. Suffixes (បច្ច័យបទ - ប្តូរថ្នាក់ពាក្យ)

| Suffix | ថ្នាក់ពាក្យដែលបង្កើតបាន (Target Class) | ឧទាហរណ៍ (Examples) |
| :--- | :--- | :--- |
| **-er / -or** | Noun (អ្នកធ្វើសកម្មភាព) | develop -> develop**er**, compile -> compil**er** |
| **-tion / -ment** | Noun (សកម្មភាព / ស្ថានភាព) | connect -> connec**tion**, develop -> develop**ment** |
| **-able / -ible** | Adjective (អាចធ្វើបាន) | access -> access**ible**, read -> read**able** |
| **-ify / -ize** | Verb (ធ្វើឱ្យទៅជា) | simple -> simpl**ify**, modern -> modern**ize** |
| **-ly** | Adverb (របៀប ឬរបៀបធ្វើ) | quick -> quick**ly**, automatic -> automatical**ly** |
