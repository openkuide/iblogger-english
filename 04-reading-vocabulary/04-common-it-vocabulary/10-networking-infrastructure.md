## 🌐 Part 1: Networking Basics & Protocol (Words 1–25)

| # | Word | Tense / Form | Khmer Meaning | Example Sentence |
|---|------|-------------|---------------|-----------------|
| 1 | **protocol** | n. (protocols) | ពិធីការទំនាក់ទំនង | HTTP is the *protocol* used for web pages. |

| 5 | **switch** | n. (switches) | ឧបករណ៍ចែកចាយបណ្តាញក្នុងស្រុក | Connect all office computers to the network *switch*. |
| 6 | **IP address** | n. (IP addresses) | អាសយដ្ឋាន IP | Every device on a network needs an *IP address*. |
| 7 | **MAC address** | n. (MAC addresses) | អាសយដ្ឋានរូបវន្ត (ម៉ាស៊ីន) | The router filters connections based on *MAC address*. |
| 8 | **subnet** | n. (subnets) | បណ្តាញរង | The database is isolated in a private *subnet*. |
| 9 | **gateway** | n. (gateways) | ច្រកទ្វារបណ្តាញ | The default *gateway* IP is 192.168.1.1. |
| 10 | **DNS** | abbr. (Domain Name System) | ប្រព័ន្ធបកប្រែឈ្មោះដែន | *DNS* translates domain names to IP addresses. |
| 11 | **port** | n. (ports) | ច្រកទិន្នន័យ | SSH uses *port* 22 by default. |
| 12 | **socket** | n. (sockets) | ចំណុចតភ្ជាប់បណ្តាញក្នុងកូដ | The app establishes a WebSocket *socket* connection. |
| 13 | **TCP** | abbr. | ពិធីការគ្រប់គ្រងការបញ្ជូន | *TCP* is connection-oriented and reliable. |
| 14 | **UDP** | abbr. | ពិធីការបញ្ជូនទិន្នន័យរហ័ស | *UDP* is used for video streaming and gaming. |
| 15 | **HTTP** | abbr. | ពិធីការផ្ទេរអត្ថបទ | *HTTP* request/response cycle is stateless. |
| 16 | **HTTPS** | abbr. | ពិធីការផ្ទេរអត្ថបទមានសុវត្ថិភាព | *HTTPS* encrypts traffic using SSL/TLS. |
| 17 | **SSL/TLS** | abbr. | ស្រទាប់សុវត្ថិភាពតភ្ជាប់ | Renew the *SSL/TLS* certificate immediately. |
| 18 | **firewall** | n. (firewalls) | របាំងការពារបណ្តាញ | The *firewall* blocked port 8080 traffic. |
| 19 | **proxy** | n. (proxies) | proxy (ម៉ាស៊ីនតំណាង) | Configure the client *proxy* settings. |
| 20 | **reverse proxy** | n. (proxies) | reverse proxy | Nginx is commonly used as a *reverse proxy*. |
| 21 | **load balancer** | n. (load balancers) | ឧបករណ៍បែងចែកបន្ទុក | The *load balancer* handles incoming HTTPS requests. |
| 22 | **bandwidth** | n. | កម្រិតបញ្ជូនទិន្នន័យ | We need more *bandwidth* for hosting services. |
| 23 | **latency** | n. | ភាពយឺត / ការរង់ចាំ | Fiber optic connections offer low *latency*. |
| 24 | **ping** | v. (pinged) / n. | ម៉ាស៊ីនសាកល្បងល្បឿនបណ្តាញ | *Ping* the server to see if it is online. |
| 25 | **throughput** | n. | បរិមាណទិន្នន័យឆ្លងកាត់ | The network *throughput* reached 1 Gbps. |

## 🗣️ Quick Phrases for Networking & Infrastructure Discussions

| Situation | English Phrase | Khmer |
|-----------|---------------|-------|
| Requesting IP whitelist | "Please whitelist my public IP address." | "សូមជួយ Whitelist public IP address របស់ខ្ញុំផង។" |
| Diagnosing network drop | "I'm running a traceroute to see where the connection drops." | "ខ្ញុំកំពុងដំណើរការ traceroute ដើម្បីមើលថាតើការតភ្ជាប់ដាច់នៅត្រង់ណា។" |
| Suggesting server upgrade | "Let's scale up the instance; we are running out of CPU." | "ចូរយើងបង្កើនទំហំម៉ាស៊ីន (scale up) instance នេះទៅ យើងជិតអស់ CPU ហើយ។" |
| Explaining DNS propagation | "DNS propagation can take up to 24 hours." | "ការបកប្រែ DNS (DNS propagation) អាចចំណាយពេលរហូតដល់ ២៤ម៉ោង។" |
| Checking server response | "Ping the staging gateway to check the network latency." | "Ping ទៅកាន់ staging gateway ដើម្បីពិនិត្យមើល network latency។" |
