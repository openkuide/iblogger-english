## 🗣️ Quick Phrases for AI Discussions

| Situation | English Phrase | Khmer |
|-----------|---------------|-------|
| Describing a model | "We're using a fine-tuned LLM for this." | "យើងប្រើ LLM ដែលបាន fine-tune សម្រាប់ការនេះ។" |
| Reporting accuracy | "The model hit 92% accuracy on the test set." | "ម៉ូដែលបានដល់ 92% accuracy លើ test set ។" |
| Discussing a problem | "We're seeing drift in production data." | "យើងឃើញ drift នៅក្នុងទិន្នន័យ production ។" |
| Proposing a solution | "Let's add more training data and retrain." | "ចូរបន្ថែមទិន្នន័យ ហើយ retrain ម្ដងទៀត។" |
| Reviewing results | "The baseline beats our current model." | "Baseline ឈ្នះម៉ូដែលបច្ចុប្បន្នរបស់យើង។" |
