# Common IT Vocabulary (កម្រងវាក្យសព្ទព័ត៌មានវិទ្យាទូទៅ)

កម្រងវាក្យសព្ទសំខាន់ៗ និងពេញនិយមបំផុតក្នុងវិស័យ IT ដែលត្រូវបានបែងចែកទៅតាមផ្នែកនីមួយៗ ដើម្បីងាយស្រួលសិក្សា និងយកទៅប្រើប្រាស់ក្នុងការសរសេរ ឬការប្រាស្រ័យទាក់ទងការងារប្រចាំថ្ងៃ។

A comprehensive list of highly frequent developer and IT vocabulary words, organized by category with definitions, parts of speech, and code-relevant examples.

## 📂 ១. Backend & Programming (ផ្នែកប្រព័ន្ធខាងក្រោយ និងការសរសេរកូដ)

| Word (ពាក្យ) | POS | អត្ថន័យជាភាសាខ្មែរ (Khmer Meaning) | ឧទាហរណ៍ក្នុងប្រយោគ (Example) |
| :--- | :--- | :--- | :--- |
| **initialize** | Verb | កំណត់តម្លៃចាប់ផ្តើម / ចាប់ផ្តើមដំបូង | You must **initialize** the variable before using it. |
| **instantiate** | Verb | បង្កើត instance នៃ class (បង្កើត Object) | We need to **instantiate** the service helper class. |
| **inherit** | Verb | ទទួលតំណែង/បន្តវេនពី (ទម្រង់ OOP) | The `Dog` class will **inherit** methods from `Animal`. |
| **override** | Verb | សរសេរសង្កត់ពីលើ (ប្តូរមុខងារដើម) | You can **override** the default configuration helper. |
| **encapsulate** | Verb | ចងក្រង ឬលាក់បាំងទិន្នន័យ (លក្ខណៈ OOP) | Use private fields to **encapsulate** internal state. |
| **compile** | Verb | បកប្រែកូដប្រភពទៅជាកូដម៉ាស៊ីន | The compiler takes a few seconds to **compile** Java code. |
| **execute** | Verb | ដំណើរការ/អនុវត្ត (កូដ ឬពាក្យបញ្ជា) | The system will **execute** the script automatically at midnight. |
| **exception** | Noun | ករណីលើកលែង ឬកំហុសឆ្គងកំឡុងពេលរត់កូដ | The program threw a `NullPointerException` **exception**. |
| **framework** | Noun | ក្របខ័ណ្ឌការងារ (បណ្តុំបណ្ណាល័យជំនួយ) | Spring Boot is a popular Java **framework**. |
| **dependency** | Noun | ភាពអាស្រ័យ (បណ្ណាល័យ ឬ module ផ្សេងទៀតដែលត្រូវការ) | Add the Maven **dependency** to your POM file. |

## 📂 ២. Frontend & User Interface (ផ្នែកខាងមុខ និងផ្ទៃកម្មវិធី)

| Word (ពាក្យ) | POS | អត្ថន័យជាភាសាខ្មែរ (Khmer Meaning) | ឧទាហរណ៍ក្នុងប្រយោគ (Example) |
| :--- | :--- | :--- | :--- |
| **render** | Verb | បង្ហាញរូបរាង/គូរផ្ទៃកម្មវិធីលើអេក្រង់ | The browser takes time to **render** complex layouts. |
| **responsive** | Adj | ដែលឆ្លើយតបទៅនឹងទំហំអេក្រង់ទូរសព្ទ/កុំព្យូទ័រ | The web app must be fully **responsive** on mobile devices. |
| **alignment** | Noun | ការតម្រឹម (ឆ្វេង ស្តាំ កណ្តាល...) | Check the vertical **alignment** of the logo inside the header. |
| **viewport** | Noun | តំបន់បង្ហាញរូបភាព ឬផ្ទៃអេក្រង់ដែលអ្នកប្រើប្រាស់មើលឃើញ | The image size adjusts automatically to the **viewport** size. |
| **element** | Noun | ធាតុបង្ក ឬសមាសភាគផ្ទៃកម្មវិធី (HTML node) | Inspect the input **element** using Chrome DevTools. |
| **component** | Noun | សមាសភាគកូដផ្ទៃកម្មវិធីដែលអាចប្រើឡើងវិញបាន | We created a reusable button **component**. |
| **latency** | Noun | ភាពយឺតយ៉ាវ (ពេលវេលារង់ចាំផ្ញើ/ទទួលទិន្នន័យ) | Optimizing image sizes helps reduce network **latency**. |
| **interactivity**| Noun | ភាពអន្តរកម្ម (សមត្ថភាពឆ្លើយតបនឹងសកម្មភាពអ្នកប្រើ) | Adding animations improves the overall **interactivity**. |
| **fallback** | Noun | ជម្រើសបម្រុង (នៅពេលជម្រើសចម្បងមានបញ្ហា) | If the main font fails to load, use Arial as a **fallback**. |
| **embed** | Verb | បញ្ចូល/បង្កប់ (រូបភាព វីដេអូ ឬកូដ) ទៅក្នុងទំព័រ | You can **embed** a YouTube video using an iframe tag. |

## 📂 ៣. Database & Data Management (ផ្នែកមូលដ្ឋានទិន្នន័យ)

| Word (ពាក្យ) | POS | អត្ថន័យជាភាសាខ្មែរ (Khmer Meaning) | ឧទាហរណ៍ក្នុងប្រយោគ (Example) |
| :--- | :--- | :--- | :--- |
| **query** | Noun/V | សំនួរទាញទិន្នន័យ / សួររកទិន្នន័យ | The select **query** returned 100 rows. |
| **migration** | Noun | ការផ្លាស់ទី ឬការកែសម្រួល schema មូលដ្ឋានទិន្នន័យ | Run the database **migration** script to add the new tables. |
| **schema** | Noun | គ្រោងឆ្អឹង ឬរចនាសម្ព័ន្ធតារាងមូលដ្ឋានទិន្នន័យ | We modified the user table **schema** to add an email field. |
| **index** | Noun/V | សន្ទស្សន៍ (ជំនួយការស្វែងរកលឿន) / បង្កើតសន្ទស្សន៍ | Adding an **index** on the email column makes queries faster. |
| **redundancy** | Noun | ភាពស្ទួនគ្នានៃទិន្នន័យ | Database normalization reduces data **redundancy**. |
| **constraint** | Noun | លក្ខខណ្ឌបង្ខំ ឬដែនកំណត់លើទិន្នន័យ | The foreign key **constraint** prevents deleting the parent row. |
| **transaction** | Noun | ប្រតិបត្តិការទិន្នន័យ (បណ្តុំសកម្មភាពដែលត្រូវជោគជ័យទាំងអស់) | If any step fails, rollback the database **transaction**. |
| **cache** | Noun/V | អង្គចងចាំបណ្តោះអាសន្នល្បឿនលឿន / រក្សាទុកបណ្តោះអាសន្ន | Store user session details in the Redis **cache**. |
| **retrieve** | Verb | ទាញយក/នាំយកទិន្នន័យមកវិញ | The application will **retrieve** user info from the database. |
| **populate** | Verb | បញ្ចូលទិន្នន័យបំពេញទៅក្នុងតារាង ឬបញ្ជី | We use seed scripts to **populate** the database with test data. |

## 📂 ៤. DevOps, Git & Deployment (ផ្នែក DevOps និងការដាក់ឱ្យដំណើរការ)

| Word (ពាក្យ) | POS | អត្ថន័យជាភាសាខ្មែរ (Khmer Meaning) | ឧទាហរណ៍ក្នុងប្រយោគ (Example) |
| :--- | :--- | :--- | :--- |
| **deploy** | Verb | ដាក់ឱ្យដំណើរការនៅលើ server | We will **deploy** the system to production tonight. |
| **containerize** | Verb | ខ្ចប់កម្មវិធីទៅក្នុង Container (ដូចជា Docker) | We should **containerize** all our backend services. |
| **pipeline** | Noun | ខ្សែសង្វាក់ការងារស្វ័យប្រវត្តិ (CI/CD) | The CI/CD **pipeline** compiles and tests the code automatically. |
| **orchestrate** | Verb | រៀបចំ និងគ្រប់គ្រងដំណើរការ cluster ស្វ័យប្រវត្តិ | Kubernetes is used to **orchestrate** container services. |
| **rollback** | Verb/N | ត្រឡប់ទៅរកកំណែចាស់ ឬស្ថានភាពដើមវិញ | If the update fails, we must **rollback** to the previous version. |
| **credentials** | Noun | ព័ត៌មានសម្ងាត់សម្រាប់ផ្ទៀងផ្ទាត់ (Username, Password, API key) | Never store database **credentials** in public repositories. |
| **cluster** | Noun | ក្រុមនៃ servers ឬ machines ដែលដំណើរការរួមគ្នា | The application runs on a Kubernetes **cluster**. |
| **automation** | Noun | ស្វ័យប្រវត្តិនីយកម្ម (ការធ្វើការងារដោយស្វ័យប្រវត្តិ) | Test **automation** saves time and reduces human error. |
| **provision** | Verb | បង្កើត ឬរៀបចំធនធាន server/infrastructures | Terraform is used to **provision** cloud infrastructures. |
| **starvation** | Noun | ស្ថានភាពដាច់ធនធាន (server ខ្វះ memory/connections) | Thread **starvation** caused the server to stop responding. |

## 📂 ៥. Agile, Standups & Collaboration (ផ្នែក Agile និងការសហការ)

| Word (ពាក្យ) | POS | អត្ថន័យជាភាសាខ្មែរ (Khmer Meaning) | ឧទាហរណ៍ក្នុងប្រយោគ (Example) |
| :--- | :--- | :--- | :--- |
| **blocker** | Noun | បញ្ហារារាំងដែលធ្វើឱ្យមិនអាចបន្តការងារបាន | Do you have any **blocker** preventing you from coding? |
| **backlog** | Noun | បញ្ជីការងារដែលត្រូវធ្វើនៅពេលក្រោយ | We have fifty features sitting in the product **backlog**. |
| **sprint** | Noun | វគ្គការងារខ្លី (ជាទូទៅមានរយៈពេល ១ ទៅ ៤ សប្តាហ៍) | We need to finish these ten tickets in this **sprint**. |
| **ticket** | Noun | កាតការងារ ឬភារកិច្ចដែលត្រូវបានចាត់ចែងក្នុងប្រព័ន្ធ (Jira) | I am working on the database refactoring **ticket**. |
| **estimation** | Noun | ការវាយតម្លៃ ឬប៉ាន់ស្មានពេលវេលា/កម្លាំងពលកម្ម | Provide a time **estimation** for this new feature setup. |
| **refactor** | Verb | កែសម្រួលកូដឱ្យស្អាត/ប្រសើរឡើងដោយមិនប្តូរមុខងារដើម | We should **refactor** this class to make it more readable. |
| **review** | Verb/N | ពិនិត្យឡើងវិញ / ការពិនិត្យឡើងវិញ | Please **review** my Pull Request when you are free. |
| **milestone** | Noun | ចំណុចដៅសំខាន់ ឬដំណាក់កាលសម្រេចបានក្នុងគម្រោង | Completing the auth system is a key **milestone**. |
| **standup** | Noun | កិច្ចប្រជុំប្រចាំថ្ងៃខ្លីៗដើម្បីរាយការណ៍ការងារ | We share our updates daily in the morning **standup**. |
| **scope** | Noun/V | វិសាលភាពការងារ / ដែនកំណត់សកម្មភាព | Adding payment is out of the **scope** of this sprint. |

| Word (ពាក្យ) | POS | អត្ថន័យជាភាសាខ្មែរ (Khmer Meaning) | ឧទាហរណ៍ក្នុងប្រយោគ (Example) |
| :--- | :--- | :--- | :--- |

| **seamless** | Adj | ដែលរលូនឥតទាស់ ឬគ្មានថ្នេរ (គ្មានបញ្ហារំខាន) | The migration provided a **seamless** upgrade experience. |
| **vulnerable** | Adj | ដែលងាយរងគ្រោះ ឬមានចន្លោះប្រហោងសុវត្ថិភាព | Using outdated libraries makes your application **vulnerable**. |
| **orthogonal** | Adj | ដែលឯករាជ្យពីគ្នា (កែចំណុចមួយ មិនប៉ះពាល់ចំណុចមួយទៀត) | Keep your system modules as **orthogonal** as possible. |
