# Software Testing & QA Vocabulary (វាក្យសព្ទ Software Testing & QA)

100 essential words for Testing, Quality Assurance, and Bug tracking discussions in IT teams.

## 🧪 Part 1: Core Testing Types (Words 1–25)

| # | Word | Tense / Form | Khmer Meaning | Example Sentence |
|---|------|-------------|---------------|-----------------|
| 1 | **QA** | abbr. (Quality Assurance) | ធានាគុណភាព | The *QA* team approved the release. |

| 3 | **automation** | n. | ស្វ័យប្រវត្តិកម្ម | *Automation* reduces testing time. |
| 4 | **unit test** | n. (unit tests) / v. | ការសាកល្បងថ្នាក់កូដតូចៗ | Write a *unit test* for this helper function. |
| 5 | **integration test** | n. (integration tests) / v. | ការសាកល្បងការផ្សារភ្ជាប់ | The *integration test* checks the DB connection. |
| 6 | **end-to-end** | adj. (E2E) | ពីចុងម្ខាងទៅចុងម្ខាងទៀត (គ្រប់ដណ្តប់ទាំងស្រុង) | Run *end-to-end* tests to simulate user flows. |
| 7 | **regression** | n. | ការសាកល្បងឡើងវិញ (ការពារកុំឱ្យខូចកូដចាស់) | Run *regression* tests before deploying. |
| 8 | **smoke test** | n. (smoke tests) | ការសាកល្បងបឋម | The *smoke test* passed in five minutes. |
| 9 | **sanity test** | n. (sanity tests) | ការសាកល្បងសមហេតុផល | Do a *sanity test* on the hotfix. |
| 10 | **performance** | n. | ដំណើរការ / ប្រសិទ្ធភាព | Run *performance* tests to check load speeds. |
| 11 | **load test** | n. (load tests) / v. | ការសាកល្បងផ្ទុកទិន្នន័យច្រើន | We *load tested* the API with 10k virtual users. |
| 12 | **stress test** | n. (stress tests) / v. | ការសាកល្បងក្រោមសម្ពាធខ្លាំង | *Stress test* the system until it crashes. |
| 13 | **usability** | n. | ភាពងាយស្រួលប្រើប្រាស់ | *Usability* testing helps improve UI layout. |
| 14 | **compatibility** | n. | ភាពអាចត្រូវគ្នាបាន (ជាមួយប្រព័ន្ធផ្សេង) | Test app *compatibility* on Safari and Chrome. |
| 15 | **security test** | n. (tests) | ការសាកល្បងសុវត្ថិភាព | Perform a *security test* on the auth module. |
| 16 | **penetration** | n. | ការជ្រៀតចូល | Conduct *penetration* testing on the server. |
| 17 | **exploratory** | adj. | ដែលរុករកនិងស្វែងយល់ | Spend two hours on *exploratory* testing. |
| 18 | **acceptance** | n. | ការយល់ព្រមទទួលយក | User *acceptance* testing (UAT) starts tomorrow. |
| 19 | **alpha test** | n. (alpha tests) | ការសាកល្បងផ្ទៃក្នុង | *Alpha test* the application with internal staff. |
| 20 | **beta test** | n. (beta tests) | ការសាកល្បងក្រៅផ្លូវការពីអ្នកប្រើប្រាស់ | Release the *beta test* version to 100 users. |
| 21 | **A/B testing** | n. | ការសាកល្បងប្រៀបធៀប A/B | We used *A/B testing* for the checkout button color. |
| 22 | **black-box** | adj. | ប្រអប់ខ្មៅ (សាកល្បងដោយមិនមើលកូដ) | *Black-box* testing focuses on inputs and outputs. |
| 23 | **white-box** | adj. | ប្រអប់ស (សាកល្បងដោយមើលរចនាសម្ព័ន្ធកូដ) | *White-box* testing requires knowledge of the code. |
| 24 | **grey-box** | adj. | ប្រអប់ប្រផេះ (សាកល្បងដោយដឹងកូដខ្លះៗ) | *Grey-box* testers inspect backend logs. |
| 25 | **functional** | adj. | ដែលទាក់ទងនឹងមុខងារ | Verify the *functional* requirements of the page. |

## 🐛 Part 2: Bug & Issue Tracking (Words 26–50)

| # | Word | Tense / Form | Khmer Meaning | Example Sentence |
|---|------|-------------|---------------|-----------------|
| 26 | **defect** | n. (defects) | គុណវិបត្តិ / បញ្ហាកូដ | Report any *defect* found in production. |
| 27 | **bug** | n. (bugs) | បញ្ហាកូដ / កំហុសបច្ចេកទេស | I found a minor *bug* in the login form. |
| 28 | **reproduce** | v. (reproduced, reproducing) | ធ្វើឱ្យកើតឡើងឡើងវិញ (បង្ហាញកំហុស) | Can you *reproduce* the issue on your machine? |
| 29 | **steps to reproduce**| n. | ជំហានដើម្បីបង្កើតកំហុសឡើងវិញ | Write clear *steps to reproduce* in the bug report. |
| 30 | **severity** | n. | កម្រិតប៉ះពាល់ (ធ្ងន់ធ្ងរ) | The bug has a *severity* level of 'Critical'. |

| 39 | **false negative** | n. (negatives) | ការរាយការណ៍ខុស (មានកំហុសតែប្រាប់ថាគ្មាន) | Avoid *false negatives* in safety-critical code. |

| 41 | **stack trace** | n. (traces) | គន្លងហៅកូដពេលមានបញ្ហា | Copy the *stack trace* into the ticket. |
| 42 | **breakpoint** | n. (breakpoints) | ចំណុចផ្អាកសម្រាប់ពិនិត្យកូដ | Set a *breakpoint* in the controller. |
| 43 | **debugger** | n. (debuggers) | កម្មវិធីដោះស្រាយបញ្ហាកូដ | Use the browser *debugger* to step through code. |
| 44 | **issue** | n. (issues) / v. | បញ្ហា / ដំណើរការបញ្ហា | Create an *issue* on GitHub. |

| 48 | **resolved** | adj. / v. | ត្រូវបានដោះស្រាយ | The database issue is now *resolved*. |
| 49 | **unresolved** | adj. | ដែលមិនទាន់ដោះស្រាយ | There are three *unresolved* bugs left. |

## 🗣️ Quick Phrases for Testing & QA Discussions

| Situation | English Phrase | Khmer |
|-----------|---------------|-------|
| Requesting reproduction | "Can you share the steps to reproduce this bug?" | "តើអ្នកអាចចែករំលែកជំហានដើម្បីបង្កើត bug នេះឡើងវិញបានទេ?" |
| Informing about test result | "The integration test failed on the staging environment." | "ការសាកល្បង integration បានបរាជ័យនៅលើ staging environment ។" |
| Suggesting mock data | "Let's mock the third-party payment gateway." | "ចូរយើងធ្វើត្រាប់ (mock) payment gateway ខាងក្រៅ។" |
| Pointing to regression | "This bug is a regression from yesterday's release." | "Bug នេះគឺជា regression ពីការ Release ម្សិលមិញ។" |
| Confirming green build | "All test suites passed, we are ready for sign-off." | "គ្រប់ test suites ទាំងអស់បានឆ្លងកាត់ យើងត្រៀមខ្លួនសម្រាប់ sign-off ហើយ។" |
