## 🗣️ Quick Phrases for Cloud & Security Discussions

| Situation | English Phrase | Khmer |
|-----------|---------------|-------|
| Recommending cloud scaling | "We should configure auto-scaling for this instance." | "យើងគួរតែកំណត់រចនាសម្ព័ន្ធ auto-scaling សម្រាប់ instance នេះ។" |
| Discussing a security fix | "We need to patch this vulnerability immediately." | "យើងត្រូវជួសជុលចំណុចខ្សោយ (vulnerability) នេះជាបន្ទាន់។" |
| Pointing to authentication | "All endpoints require authentication tokens." | "គ្រប់ endpoints ទាំងអស់ត្រូវការ authentication tokens ។" |
| Explaining backups | "The database backups are scheduled daily." | "ការ Backup ទិន្នន័យត្រូវបានកំណត់ធ្វើឡើងរាល់ថ្ងៃ។" |
| Pointing out access rights | "Ensure the user role follows the principle of least privilege." | "ធានាថាតួនាទីរបស់អ្នកប្រើប្រាស់អនុវត្តតាមគោលការណ៍ least privilege ។" |
