# Sentence Structure (ទម្រង់ប្រយោគ)

ការយល់ដឹងពីរបៀបផ្គុំពាក្យចូលគ្នាដើម្បីបង្កើតជាប្រយោគ គឺជាគន្លឹះដំបូងក្នុងការសរសេរ និងនិយាយភាសាអង់គ្លេសប្រកបដោយភាពច្បាស់លាស់ និងវិជ្ជាជីវៈ។

Understanding how to put words together to form sentences is key to writing and speaking English with clarity and professionalism.

---

## 🧱 ធាតុផ្សំចម្បងនៃប្រយោគ (Core Elements of a Sentence)

ប្រយោគនីមួយៗជាទូទៅត្រូវការធាតុផ្សំសំខាន់ៗពីរ៖
1. **Subject (ប្រធាន):** អ្នកធ្វើសកម្មភាព ឬជាប្រធានបទនៃប្រយោគ។
2. **Predicate (កិរិយា/កិរិយាឃ្លា):** សកម្មភាព ឬអ្វីដែលត្រូវបាននិយាយអំពីប្រធាន។

```
[Subject (ប្រធាន)] + [Verb (កិរិយាសព្ទ)] + [Object/Complement (កម្មបទ/ពាក្យបង្គ្រប់)]
```
* ឧទាហរណ៍៖ **The system** (Subject) + **sends** (Verb) + **an email alert** (Object).

## ⚙️ ប្រភេទប្រយោគទាំង ៤ (The 4 Types of Sentence Structures)

ដើម្បីឱ្យការសរសេររបស់យើងមានភាពរស់រវើក យើងត្រូវចេះប្រើប្រាស់ប្រភេទប្រយោគផ្សេងៗគ្នា៖

### ១. Simple Sentence (ប្រយោគទោល)
មាន Clause ឯករាជ្យ (Independent Clause) តែមួយគត់ (មានន័យគ្រប់គ្រាន់ និងអាចឈរតែឯងបាន)។
* *ឧទាហរណ៍៖* The server crashed. (ម៉ាស៊ីនបម្រើបានគាំង។)

### ២. Compound Sentence (ប្រយោគចម្រុះ-សមាសភាព)
មាន Clause ឯករាជ្យពីរ ឬច្រើន ភ្ជាប់គ្នាដោយឈ្នាប់ **FANBOYS** (For, And, Nor, But, Or, Yet, So) ឬសញ្ញា Semicolon (`;`)។
* *ឧទាហរណ៍៖* The frontend is ready, **but** the backend is still database-locked. (ផ្នែកខាងមុខគឺរួចរាល់ហើយ ប៉ុន្តែផ្នែកខាងក្រោយនៅតែជាប់សោទិន្នន័យ។)

### ៣. Complex Sentence (ប្រយោគចម្រុះ-ល្បះរង)
មាន Clause ឯករាជ្យមួយ (Independent Clause) និង Clause មិនឯករាជ្យ (Dependent Clause) យ៉ាងហោចណាស់មួយ ដែលភ្ជាប់គ្នាដោយឈ្នាប់ដូចជា `because`, `although`, `if`, `when`, `since`...។
* *ឧទាហរណ៍៖* **Although** the code has no compile errors, it fails the unit tests. (ទោះបីជាកូដគ្មានកំហុស compile ក៏ដោយ វាបានធ្លាក់ការធ្វើតេស្ត unit tests។)

### ៤. Compound-Complex Sentence (ប្រយោគចម្រុះស្មុគស្មាញ)
មាន Clause ឯករាជ្យយ៉ាងហោចណាស់ពីរ និង Clause មិនឯករាជ្យយ៉ាងហោចណាស់មួយ។
* *ឧទាហរណ៍៖* Because the network latency was high, the API request timed out, and the user interface showed an error. (ដោយសារតែភាពយឺតយ៉ាវនៃបណ្តាញខ្ពស់ ការទាមទារ API ហួសពេលកំណត់ ហើយផ្ទៃកម្មវិធីអ្នកប្រើប្រាស់បានបង្ហាញកំហុស។)

## ⚠️ កំហុសឆ្គងទូទៅដែលត្រូវចៀសវាង (Common Mistakes to Avoid)

* **Run-on Sentences:** ការសរសេរប្រយោគឯករាជ្យពីរជាប់គ្នាដោយគ្មានឈ្នាប់ ឬសញ្ញាខណ្ឌសមស្រប។
  * *ខុស៖* The developer wrote the patch he deployed it immediately.
  * *ត្រូវ៖* The developer wrote the patch, **and** he deployed it immediately.
* **Sentence Fragments:** ប្រយោគដែលមិនពេញលេញ (គ្មានប្រធាន ឬកិរិយាសព្ទ ឬមានតែ dependent clause ឈរតែឯង)។
  * *ខុស៖* Because the database connection pool was full.
  * *ត្រូវ៖* The API returned a 500 status **because the database connection pool was full.**
