# Essential English Tenses (កាលសំខាន់ៗទាំង ១២)

នៅក្នុងភាសាអង់គ្លេស កាល (Tenses) ត្រូវបានប្រើដើម្បីបង្ហាញពីពេលវេលានៃសកម្មភាពមួយ (អតីតកាល បច្ចុប្បន្នកាល ឬអនាគតកាល)។ ទោះបីជាមានកាលរហូតដល់ ១២ ក៏ដោយ ក៏កាលចំនួន ៥ ខាងក្រោមនេះត្រូវបានប្រើប្រាស់ច្រើនជាងគេបំផុត (ប្រហែល ៨០% ទៅ ៩០% នៃការប្រើប្រាស់ប្រចាំថ្ងៃ និងក្នុងការសរសេរឯកសារបច្ចេកទេស)។

In English, tenses indicate the time of an action. Although there are 12 tenses, the following 5 tenses are the most commonly used, representing 80% to 90% of everyday and technical communication.

---

## ⭐️ កាលទាំង ៥ ដែលប្រើញឹកញាប់បំផុត (The 5 Most Common Tenses)

### ១. Present Simple (បច្ចុប្បន្នកាលធម្មតា)
* **ទម្រង់ (Structure):** `Subject + Verb (s/es)`
* **ការប្រើប្រាស់ (Usage):** ប្រើសម្រាប់ការពិតទូទៅ ទម្លាប់ប្រចាំថ្ងៃ ឬស្ថានភាពអចិន្ត្រៃយ៍។ នៅក្នុង IT វាត្រូវបានប្រើដើម្បីពណ៌នាពីរបៀបដែលប្រព័ន្ធ ឬកូដដំណើរការ។
* **ឧទាហរណ៍៖**
  - The compiler **translates** source code into machine code. (អ្នកបកប្រែកូដ បកប្រែកូដប្រភពទៅជាកូដម៉ាស៊ីន។)
  - We **use** Git for version control. (ពួកយើងប្រើប្រាស់ Git សម្រាប់ការគ្រប់គ្រងកំណែកូដ។)

### ២. Past Simple (អតីតកាលធម្មតា)
* **ទម្រង់ (Structure):** `Subject + Verb (V2 / -ed)`
* **ការប្រើប្រាស់ (Usage):** ប្រើសម្រាប់សកម្មភាពដែលបានកើតឡើង និងបញ្ចប់សព្វគ្រប់ក្នុងអតីតកាល។ ប្រើញឹកញាប់ក្នុងរបាយការណ៍ការងារ ឬ commit messages។
* **ឧទាហរណ៍៖**
  - I **fixed** the memory leak yesterday. (ខ្ញុំបានជួសជុលបញ្ហាលេចធ្លាយអង្គចងចាំកាលពីម្សិលមិញ។)
  - The team **released** version 2.0 last week. (ក្រុមការងារបានចេញផ្សាយកំណែ ២.០ កាលពីសប្តាហ៍មុន។)

### ៣. Present Perfect (បច្ចុប្បន្នកាលបរិបូណ៌)
* **ទម្រង់ (Structure):** `Subject + have/has + Past Participle (V3)`
* **ការប្រើប្រាស់ (Usage):** ប្រើសម្រាប់សកម្មភាពដែលបានកើតឡើងក្នុងអតីតកាល ប៉ុន្តែមានផលប៉ះពាល់ ឬលទ្ធផលមកដល់បច្ចុប្បន្ន ឬទើបតែបញ្ចប់ថ្មីៗ។
* **ឧទាហរណ៍៖**
  - We **have deployed** the application to staging. (ពួកយើងបានដាក់ឱ្យដំណើរការកម្មវិធីនៅលើ staging រួចហើយ។)
  - **Have** you **updated** the documentation yet? (តើអ្នកបានធ្វើបច្ចុប្បន្នភាពឯកសារណែនាំហើយឬនៅ?)

### ៤. Present Continuous (បច្ចុប្បន្នកាលកំពុងបន្ត)
* **ទម្រង់ (Structure):** `Subject + am/is/are + Verb-ing`
* **ការប្រើប្រាស់ (Usage):** ប្រើសម្រាប់សកម្មភាពដែលកំពុងកើតឡើងនៅពេលនិយាយ ឬគម្រោងបច្ចុប្បន្ន។
* **ឧទាហរណ៍៖**
  - The server **is running** in debug mode. (ម៉ាស៊ីនបម្រើកំពុងដំណើរការក្នុងរបៀប debug។)
  - I **am working** on the database migration today. (ខ្ញុំកំពុងធ្វើការលើការផ្លាស់ប្តូរមូលដ្ឋានទិន្នន័យនៅថ្ងៃនេះ។)

### ៥. Future Simple (អនាគតកាលធម្មតា)
* **ទម្រង់ (Structure):** `Subject + will + Verb (infinitive)`
* **ការប្រើប្រាស់ (Usage):** ប្រើសម្រាប់សកម្មភាពដែលនឹងកើតឡើងក្នុងអនាគត ការសន្យា ឬការសម្រេចចិត្តភ្លាមៗ។
* **ឧទាហរណ៍៖**
  - We **will deploy** the bug fix tomorrow morning. (ពួកយើងនឹងដាក់ bug fix ឱ្យដំណើរការនៅព្រឹកស្អែក។)
  - The system **will automatically restart** after the update. (ប្រព័ន្ធនឹងចាប់ផ្តើមឡើងវិញដោយស្វ័យប្រវត្តិកោក្រោយពេលធ្វើបច្ចុប្បន្នភាព។)

## 📋 តារាងសង្ខេបកាលទាំង ១២ (Summary Table of All 12 Tenses)

| Tense Aspect | Present (បច្ចុប្បន្ន) | Past (អតីត) | Future (អនាគត) |
| :--- | :--- | :--- | :--- |
| **Simple** | I code daily. | I coded yesterday. | I will code tomorrow. |
| **Continuous** | I am coding now. | I was coding when it crashed. | I will be coding at 10 PM. |
| **Perfect** | I have coded the feature. | I had coded before the meeting. | I will have coded it by Friday. |
| **Perfect Continuous**| I have been coding for 3 hours. | I had been coding since noon. | I will have been coding for hours. |

## 💾 កិរិយាសព្ទ IT ប្រើប្រាស់ញឹកញាប់ និងកាល (Common IT Verbs & Tenses)

ខាងក្រោមនេះជាបញ្ជីកិរិយាសព្ទសំខាន់ៗក្នុងវិស័យ IT ដែលរៀបចំជាបីទម្រង់ (V1, V2, V3) សម្រាប់ប្រើប្រាស់ក្នុងកាលផ្សេងៗគ្នា៖

| Base Form (V1 - បច្ចុប្បន្ន) | Past Simple (V2 - អតីត) | Past Participle (V3 - បរិបូណ៌) | អត្ថន័យ (Meaning) | ឧទាហរណ៍ការប្រើប្រាស់ (Example) |
| :--- | :--- | :--- | :--- | :--- |
| **build** | built | built | សាងសង់/បង្កើត | We **build** the code. / I **built** it yesterday. |
| **run** | ran | run | ដំណើរការ/រត់ | The script **runs**. / I **ran** the command. / The build has **run**. |
| **write** | wrote | written | សរសេរ | She **writes** tests. / He **wrote** comments. / It has been **written**. |
| **find** | found | found | រកឃើញ | We **find** bugs. / I **found** a memory leak. |
| **deploy** | deployed | deployed | ដាក់ឱ្យដំណើរការ | They **deploy** weekly. / We **deployed** to staging. |
| **fix** | fixed | fixed | ជួសជុល | **Fix** the bug. / I **fixed** the compilation error. |
| **debug** | debugged | debugged | ស្វែងរកកំហុស | She **debugs** the code. / He **debugged** the thread starvation. |
| **commit** | committed | committed | រក្សាទុកបម្រែបម្រួល | Please **commit** changes. / I **committed** the patch. |
| **merge** | merged | merged | បញ្ចូលគ្នា | **Merge** the PR. / The branches were **merged**. |
| **configure** | configured | configured | កំណត់រចនាសម្ព័ន្ធ | **Configure** the port. / We **configured** the database pool. |
| **implement** | implemented | implemented | អនុវត្ត/បង្កើត | **Implement** the API. / We **implemented** JWT login. |
| **optimize** | optimized | optimized | បង្កើនប្រសិទ្ធភាព | **Optimize** queries. / The JVM **optimized** execution speed. |
