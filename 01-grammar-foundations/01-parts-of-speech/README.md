# Parts of Speech (ថ្នាក់នៃពាក្យ)

នៅក្នុងភាសាអង់គ្លេស ពាក្យទាំងឡាយត្រូវបានបែងចែកជា ៨ ក្រុមធំៗ ទៅតាមមុខងាររបស់វានៅក្នុងប្រយោគ។ ការយល់ដឹងពីថ្នាក់នៃពាក្យជួយឱ្យយើងស្ថាបនាប្រយោគបានត្រឹមត្រូវតាមវេយ្យាករណ៍។

In English, words are categorized into 8 main groups based on their function within a sentence. Understanding parts of speech helps us build grammatically correct sentences.

---

## 📊 ថ្នាក់នៃពាក្យទាំង ៨ (The 8 Parts of Speech)

| ថ្នាក់ពាក្យ (Part of Speech) | មុខងារ (Function) | ឧទាហរណ៍ពាក្យ (Examples) | ឧទាហរណ៍ក្នុងប្រយោគ (Example Sentences) |
| :--- | :--- | :--- | :--- |
| **Noun (នាម)** | សំដៅលើ មនុស្ស ទីកន្លែង វត្ថុ ឬគំនិត | developer, computer, Phnom Penh, freedom | The **developer** bought a new **computer**. |
| **Pronoun (សព្វនាម)** | ប្រើជំនួសនាម ដើម្បីកុំឱ្យជាន់គ្នា | I, you, he, she, it, we, they | **She** loves coding in Java. |
| **Verb (កិរិយាសព្ទ)** | បង្ហាញសកម្មភាព ឬស្ថានភាព | write, compile, run, is, exist | We **run** the code on the server. |
| **Adjective (គុណនាម)** | ពណ៌នា ឬបញ្ជាក់ន័យឱ្យនាម/សព្វនាម | fast, clear, bug-free, modern | This is a **modern** framework. |
| **Adverb (គុណកិរិយា)** | បញ្ជាក់ន័យឱ្យកិរិយា គុណនាម ឬគុណកិរិយាផ្សេងទៀត | quickly, carefully, extremely | The application runs **extremely** fast. |
| **Preposition (ធ្នាក់)** | បង្ហាញទំនាក់ទំនងរវាងនាម និងពាក្យដទៃ (ទីកន្លែង ពេលវេលា...) | in, on, at, under, through, with | The configuration file is **on** the desktop. |
| **Conjunction (ឈ្នាប់)** | ភ្ជាប់ពាក្យ ឃ្លា ឬប្រយោគចូលគ្នា | and, but, or, because, although | I wrote the code, **but** it has syntax errors. |
| **Interjection (ឧទានសព្ទ)** | បង្ហាញអារម្មណ៍រំភើប ភ្ញាក់ផ្អើល ឬអារម្មណ៍ខ្លាំង | Oh!, Wow!, Alas!, Oops! | **Oops!** I deleted the wrong database table. |

## 💡 គន្លឹះសំខាន់ៗសម្រាប់ការសិក្សា (Key Tips for Learning)

1. **ពាក្យមួយអាចមានថ្នាក់ពាក្យច្រើន (A word can be more than one part of speech):**
   - ឧទាហរណ៍៖ ពាក្យ **"run"** អាចជា **Verb** (I **run** the build script.) ឬជា **Noun** (This is a successful **run**.)។
   - ចូរពិនិត្យមើលមុខងាររបស់វានៅក្នុងប្រយោគជានិច្ច។

2. **ការប្រើប្រាស់ក្នុងវិស័យ IT (IT Usage Context):**
   - **Nouns**: `class`, `variable`, `object`, `database`, `server`.
   - **Verbs**: `initialize`, `debug`, `deploy`, `inherit`, `override`.
   - **Adjectives**: `deprecated`, `synchronous`, `asynchronous`, `immutable`.
