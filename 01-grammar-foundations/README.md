# Module 1: Grammar Foundations (គ្រឹះនៃវេយ្យាករណ៍)

សិក្សាពីគ្រឹះដ៏សំខាន់នៃភាសាអង់គ្លេស រួមមានថ្នាក់នៃពាក្យ ទម្រង់ប្រយោគ និងកាលសំខាន់ៗ ដើម្បីកសាងមូលដ្ឋានគ្រឹះដ៏រឹងមាំសម្រាប់ជំនាញសរសេរ និងនិយាយ។

Learn the essential building blocks of English grammar, including parts of speech, sentence structures, and crucial tenses to build a solid foundation for writing and speaking.
