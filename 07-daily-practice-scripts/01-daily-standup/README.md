# Daily Standup Scripts (ស្គ្រីបប្រជុំ Standup ប្រចាំថ្ងៃ)

សូមទន្ទេញចាំ និងហាត់និយាយប្រយោគ ឬកថាខណ្ឌខាងក្រោមឱ្យបានស្ទាត់ជំនាញ។ អ្នកអាចផ្លាស់ប្តូរពាក្យក្នុងចន្លោះទ្រង់ទ្រាយ `[...]` ទៅតាមភារកិច្ចជាក់ស្តែងរបស់អ្នក។

Memorize and practice speaking the following paragraphs out loud. Replace the bracketed `[...]` placeholders with details from your actual tasks.

---

## 🗣️ ស្គ្រីបទី ១៖ របាយការណ៍ទូទៅប្រចាំថ្ងៃ (Standard Progress Update)

> "Yesterday, I finalized the development of the **[user login API]** and wrote unit tests for the validator service. Today, my focus will be on containerizing the backend service using **[Docker]** and pushing it to the staging repository. Currently, I have no blockers."

### 🗣️ អានជាសំឡេងខ្មែរសម្រួល (Khmer Translation)
> «កាលពីម្សិលមិញ ខ្ញុំបានធ្វើឱ្យរួចរាល់នូវការអភិវឌ្ឍ **[user login API]** និងសរសេរ unit tests សម្រាប់ validator service។ ថ្ងៃនេះ ការផ្តោតអារម្មណ៍របស់ខ្ញុំនឹងទៅលើការដាក់ backend service ឱ្យដំណើរការក្នុង container ដោយប្រើ **[Docker]** ហើយរុញវាទៅ staging repository។ បច្ចុប្បន្ន ខ្ញុំគ្មានបញ្ហាស្ទះអ្វីទេ។»

---

## 🗣️ ស្គ្រីបទី ២៖ នៅពេលមានបញ្ហាស្ទះការងារ (Reporting a Blocker)

> "Yesterday, I worked on integrating the **[third-party payment gateway]**. However, today I am blocked because the sandbox environment seems to be down, and I cannot verify the transaction response. I have reached out to their support team, but until they reply, I will switch my focus to fixing outstanding UI bugs on the **[checkout page]**."

### 🗣️ អានជាសំឡេងខ្មែរសម្រួល (Khmer Translation)
> «កាលពីម្សិលមិញ ខ្ញុំបានធ្វើការលើការបញ្ជូល **[third-party payment gateway]**។ ប៉ុន្តែ ថ្ងៃនេះខ្ញុំកំពុងជួបបញ្ហាស្ទះ ដោយសារតែ sandbox environment មើលទៅដូចជាគាំង ហើយខ្ញុំមិនអាចផ្ទៀងផ្ទាត់ transaction response បានទេ។ ខ្ញុំបានទាក់ទងទៅក្រុមជំនួយរបស់ពួកគេរួចហើយ ប៉ុន្តែរហូតដល់ពួកគេឆ្លើយតប ខ្ញុំនឹងប្តូរការផ្តោតអារម្មណ៍ទៅលើការដោះស្រាយ bugs ផ្នែក UI ដែលនៅសេសសល់លើ **[checkout page]**។»

---

## 🗣️ ស្គ្រីបទី ៣៖ នៅពេលបញ្ចប់ការងារមុនកំណត់ (Completed Task Ahead of Time)

> "Yesterday, I managed to finish the database migration script earlier than expected. I've deployed it to the development server, and everything is running smoothly. Today, I'll be picking up the next ticket on the backlog, which is **[implementing JWT token refresh logic]**."

### 🗣️ អានជាសំឡេងខ្មែរសម្រួល (Khmer Translation)
> «កាលពីម្សិលមិញ ខ្ញុំបានរៀបចំបញ្ចប់ database migration script លឿនជាងការរំពឹងទុក។ ខ្ញុំបានដាក់ឱ្យវាដំណើរការលើ development server រួចហើយ ហើយគ្រប់យ៉ាងដំណើរការយ៉ាងរលូន។ ថ្ងៃនេះ ខ្ញុំនឹងចាប់យក ticket បន្ទាប់ពី backlog ដែលជាការ **[implementing JWT token refresh logic]**។»
