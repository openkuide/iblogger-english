# Asking for Help & Debugging (ស្គ្រីបសុំជំនួយ និងការដោះស្រាយ Bug)

នៅក្នុងវិស័យ IT ការសុំជំនួយពីមិត្តរួមក្រុមដោយបង្ហាញពីអ្វីដែលអ្នកបានសាកល្បងធ្វើរួចរាល់ហើយ គឺជាសញ្ញានៃអ្នកធ្វើការអាជីព។ ចូរទន្ទេញស្គ្រីបខាងក្រោមដើម្បីសុំជំនួយប្រកបដោយប្រសិទ្ធភាព។

In IT, asking for help by explaining what you have already tried demonstrates professionalism. Memorize these scripts to request debugging support effectively.

---

## 🗣️ ស្គ្រីបទី ១៖ ការសុំជំនួយសាមញ្ញ និងគួរសម (Polite Request for Help)

> "Hi **[Sophea]**, do you have a few minutes? I've run into a strange error while configuring the **[Kubernetes cluster]**. I've been trying to debug it for the last hour, but I haven't been able to resolve it. I would really appreciate it if you could help me look over the configuration files."

### 🗣️ អានជាសំឡេងខ្មែរសម្រួល (Khmer Translation)
> «សួស្តី **[សុភារ]** តើអ្នកមានពេលពីរបីនាទីទេ? ខ្ញុំបានជួបប្រទះ error ប្លែកមួយ ពេលកំពុងកំណត់រចនាសម្ព័ន្ធ **[Kubernetes cluster]**។ ខ្ញុំបានព្យាយាម debug វាអស់រយៈពេលមួយម៉ោងហើយ ប៉ុន្តែនៅតែមិនទាន់អាចដោះស្រាយបាន។ ខ្ញុំនឹងដឹងគុណខ្លាំងណាស់ ប្រសិនបើអ្នកអាចជួយពិនិត្យមើលឯកសារកំណត់រចនាសម្ព័ន្ធទាំងនេះ។»

---

## 🗣️ ស្គ្រីបទី ២៖ ការបង្ហាញពីអ្វីដែលអ្នកបានសាកល្បងរួចហើយ (Explaining what you've tried)

> "I keep getting a **[500 Internal Server Error]** when hitting the signup endpoint. I've already verified the database credentials, checked the log files, and confirmed that the schema migration script ran successfully. Despite these checks, the connection pool still fails. Could you help me look at my Hibernate mapping config?"

### 🗣️ អានជាសំឡេងខ្មែរសម្រួល (Khmer Translation)
> «ខ្ញុំនៅតែទទួលបាន **[500 Internal Server Error]** ពេលហៅទៅ endpoint ចុះឈ្មោះគណនី។ ខ្ញុំបានផ្ទៀងផ្ទាត់ database credentials, ពិនិត្យ log files, និងបញ្ជាក់ថា schema migration script ដំណើរការជោគជ័យរួចហើយ។ ទោះជាយ៉ាងណាក៏ដោយ connection pool នៅតែធ្លាក់។ តើអ្នកអាចជួយពិនិត្យមើលការកំណត់ mapping របស់ Hibernate របស់ខ្ញុំបានទេ?»

---

## 🗣️ ស្គ្រីបទី ៣៖ ការសុំធ្វើ Pair Programming (Requesting Pair Programming)

> "Hi, I'm having some trouble setting up the **[Docker environment]** on my local machine. I'm following the README guides, but it fails to mount the volumes properly. If you have some time today, could we do a quick screen-share session to go through it together?"

### 🗣️ អានជាសំឡេងខ្មែរសម្រួល (Khmer Translation)
> «សួស្តី ខ្ញុំមានបញ្ហាមួយចំនួនក្នុងការដំឡើង **[Docker environment]** លើម៉ាស៊ីនផ្ទាល់ខ្លួនរបស់ខ្ញុំ។ ខ្ញុំកំពុងធ្វើតាមមគ្គុទ្ទេសក៍ក្នុង README ប៉ុន្តែវាមិនអាច mount volumes បានត្រឹមត្រូវទេ។ ប្រសិនបើអ្នកមានពេលនៅថ្ងៃនេះ តើយើងអាចធ្វើការចែករំលែកអេក្រង់ (screen-share) ខ្លីមួយ ដើម្បីពិនិត្យវាជាមួយគ្នាបានទេ?»
