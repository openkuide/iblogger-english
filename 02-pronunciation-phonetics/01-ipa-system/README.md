# International Phonetic Alphabet - IPA (តារាងសូរសព្ទអន្តរជាតិ)

តារាងសូរសព្ទអន្តរជាតិ (IPA) គឺជាប្រព័ន្ធសញ្ញាដែលតំណាងឱ្យសំឡេងនីមួយៗក្នុងការនិយាយភាសាអង់គ្លេស។ ការយល់ដឹងពី IPA ជួយឱ្យយើងអាចអានបញ្ចេញសំឡេងពាក្យថ្មីៗបានត្រឹមត្រូវដោយគ្រាន់តែមើលនៅក្នុងវចនានុក្រម។

The International Phonetic Alphabet (IPA) is a system of phonetic notation that represents each distinct sound in English speech. Learning IPA enables you to pronounce any new word correctly by looking up its phonetic symbols in a dictionary.

---

## 🎵 ការបែងចែកសំឡេងក្នុងភាសាអង់គ្លេស (English Sounds Division)

ភាសាអង់គ្លេសមានអក្សរ (Letters) ចំនួន ២៦ តែកាលណាផ្សំជាសំឡេង (Sounds/Phonemes) មានរហូតដល់ **៤៤ សំឡេង**៖
1. **Vowels (ស្រៈ):** ២០ សំឡេង (ចែកជា Monophthongs/ស្រៈទោល ១២ និង Diphthongs/ស្រៈផ្សំ ៨)។
2. **Consonants (ព្យញ្ជនៈ):** ២៤ សំឡេង (ចែកជា Voiced/សំឡេងមានញ័រខ្សែសំឡេង និង Voiceless/សំឡេងឥតញ័រ)។

## 📊 សំឡេងសំខាន់ៗដែលតែងតែពិបាកសម្រាប់ជនជាតិខ្មែរ (Common Challenging Sounds for Khmer Speakers)

| និមិត្តសញ្ញា IPA (Symbol) | ប្រភេទសំឡេង (Sound Type) | ឧទាហរណ៍ពាក្យ (Examples) | ការបញ្ចេញសំឡេង (Khmer Approximation / Tips) |
| :--- | :--- | :--- | :--- |
| **/θ/** | Voiceless dental fricative | **th**ink, pa**th** | ដាក់ចុងអណ្តាតនៅចន្លោះធ្មេញ រួចផ្លុំខ្យល់ចេញ (មិនមែនសំឡេង "ស" ឬ "ថ" ទេ) |
| **/ð/** | Voiced dental fricative | **th**is, **th**em | ដាក់ចុងអណ្តាតនៅចន្លោះធ្មេញ រួចបញ្ចេញសំឡេងញ័រ (មិនមែនសំឡេង "ដ" ទេ) |
| **/r/** | Post-alveolar approximant | **r**un, **r**ead | កុំអណ្តាតទៅក្រោយបន្តិច តែមិនត្រូវឱ្យប៉ះនឹងក្រអូមមាត់ឡើយ (មិនមែន "រ" ខ្មែរទេ) |
| **/ʃ/** | Voiceless postalveolar fricative | **sh**ip, **sh**ow | ធ្វើសំឡេង "ស៊ូ" ឬ "ស្ហ៊ូ" (ប្រហាក់ប្រហែលនឹងការប្រាប់គេឱ្យស្ងាត់) |
| **/tʃ/** | Voiceless postalveolar affricate | **ch**ip, **ch**urch | ស្រដៀងនឹងសំឡេង "ឆ" ខ្មែរ តែត្រូវសង្កត់ និងបញ្ចេញខ្យល់ខ្លាំងជាង |
| **/v/** | Voiced labiodental fricative | **v**iew, sa**ve** | ធ្មេញខាងលើប៉ះបបូរមាត់ខាងក្រោម រួចធ្វើសំឡេងញ័រ "វ៉ឺ" |

## 🔍 សំឡេងចុងក្រោយ (Ending Sounds)
ជនជាតិខ្មែរយើងច្រើនតែភ្លេច ឬមិនទម្លាប់បញ្ចេញសំឡេងចុងក្រោយនៃពាក្យ (Ending Sounds) ដែលនាំឱ្យមានការយល់ច្រឡំ។
* **សាកល្បង៖**
  - **Code** /koʊd/ vs **Cope** /koʊp/ vs **Coat** /koʊt/
  - **File** /faɪl/ vs **Find** /faɪnd/ vs **Fight** /faɪt/
  - **Developer** ត្រូវបញ្ចេញសំឡេង /r/ ស្រាល ឬ /ə/ នៅចុងបញ្ចប់។
