# Connected Speech & Linking (ការភ្ជាប់សំឡេងពេលនិយាយ)

ជនជាតិដើមមិននិយាយមួយពាក្យៗដាច់ពីគ្នានោះទេ។ ពួកគេភ្ជាប់ពាក្យចូលគ្នាដើម្បីឱ្យការនិយាយមានភាពរលូន និងលឿន។ ការយល់ដឹងពីបាតុភូតនេះជួយឱ្យអ្នកស្តាប់គេយល់កាន់តែច្បាស់ និងនិយាយបានធម្មជាតិជាងមុន។

Native speakers don't pronounce each word separately. They link words together to create a smooth, continuous flow of speech. Understanding this is key to improving both listening comprehension and speaking naturalness.

---

## 🔗 ១. ប្រភេទនៃការភ្ជាប់សំឡេង (Types of Linking)

### ក. ព្យញ្ជនៈភ្ជាប់ទៅស្រៈ (Consonant to Vowel - C to V)
នៅពេលពាក្យមួយបញ្ចប់ដោយសំឡេងព្យញ្ជនៈ ហើយពាក្យបន្ទាប់ចាប់ផ្តើមដោយសំឡេងស្រៈ សំឡេងទាំងពីរនឹងត្រូវភ្ជាប់គ្នា។
* *សរសេរ៖* **turn it on**
* *និយាយ៖* /tɜː.nɪ.tɒn/ -> "tur-ni-ton"
* *សរសេរ៖* **run out**
* *និយាយ៖* /rʌ.naʊt/ -> "ru-nout"
* *សរសេរ៖* **check up**
* *និយាយ៖* /tʃe.kʌp/ -> "che-kup"

### ខ. ស្រៈភ្ជាប់ទៅស្រៈ (Vowel to Vowel - V to V)
នៅពេលពាក្យមួយបញ្ចប់ដោយសំឡេងស្រៈ ហើយពាក្យបន្ទាប់ចាប់ផ្តើមដោយសំឡេងស្រៈ សំឡេងអន្តរកាល /j/ (y-sound) ឬ /w/ (w-sound) នឹងត្រូវបញ្ចូលដើម្បីឱ្យមានភាពងាយស្រួលក្នុងការនិយាយ។
* **បញ្ចូលសំឡេង /j/ (y-sound):** (ប្រសិនបើស្រៈដំបូងបញ្ចប់ដោយសំឡេង /iː/, /ɪ/, /aɪ/, /ɔɪ/)
  - **he is** -> /hiː j ɪz/ -> "he-yis"
  - **my app** -> /maɪ j æp/ -> "my-yapp"
* **បញ្ចូលសំឡេង /w/ (w-sound):** (ប្រសិនបើស្រៈដំបូងបញ្ចប់ដោយសំឡេង /uː/, /oʊ/, /aʊ/)
  - **go out** -> /ɡoʊ w aʊt/ -> "go-wout"
  - **do it** -> /duː w ɪt/ -> "do-wit"

## ✂️ ២. ការបំបាត់សំឡេង និងការកាត់ពាក្យ (Elision & Contraction)

ក្នុងការនិយាយប្រចាំថ្ងៃ សំឡេងខ្លះត្រូវបានលុបចោលទាំងស្រុងដើម្បីភាពងាយស្រួល។

* **Elision (ការលុបសំឡេង /t/ ឬ /d/ ចោល):** នៅពេលវាស្ថិតនៅចន្លោះព្យញ្ជនៈពីរផ្សេងទៀត។
  - **next week** -> និយាយថា "nex-week" (បាត់សំឡេង /t/)
  - **last night** -> និយាយថា "las-night"
  - **send configuration** -> និយាយថា "sen-configuration"
* **Contractions (ការកាត់ពាក្យ):** តែងតែប្រើក្នុងការនិយាយ និងការសរសេរក្រៅផ្លូវការ។
  - **I am** -> I'm
  - **do not** -> don't
  - **it is** -> it's

## 🏋️‍♂️ លំហាត់អនុវត្ត (Practice Exercise)
សូមសាកល្បងអានប្រយោគខាងក្រោមដោយប្រើច្បាប់ភ្ជាប់សំឡេង៖
1. *“Turn on the server.”* -> **Turn-on** the server. (tur-non)
2. *“It is an issue.”* -> **It-is-an-issue**. (I-ti-za-nish-sue)
3. *“Can I build it?”* -> **Can-I** **build-it**? (ca-I buil-dit)
